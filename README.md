# Fairplay UI Transformation

High-fidelity, mobile-responsive UI overlay for the Fairplay platform.

## Features
- **Dynamic Footer:** Automatically switches between Guest (Home, Inplay, Offers) and Authenticated (Home, Inplay, My Markets, Referral, Passbook) states.
- **Premium Register Modal:** Bottom-sheet registration form with high-fidelity branding and mobile input validation.
- **Redesigned Sidebar:** Custom-branded sidebar with "Demo" profile state and centered logout functionality.
- **Global Theme:** Consistent theme-blue (#4a90e2) branding applied across all injected components.

## Local Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```
3. Open `http://localhost:8004` in your browser.
